# 🤖 Automated Data Analyst Job Tracker — Pakistan

## 📌 Project Overview
A fully automated workflow built with n8n that finds 
live Data Analyst job openings across Pakistan every 
day — completely hands-free.

## ⚙️ How It Works
1. Runs automatically every day via Schedule Trigger
2. Calls a live Jobs API to fetch Data Analyst roles in Pakistan
3. Extracts 6 key fields per listing:
   - ✅ Job Title
   - ✅ Company Name
   - ✅ Location
   - ✅ Date Posted
   - ✅ Job Description
   - ✅ Required Skills
4. Saves all listings to Google Sheets automatically
5. Sends a daily email digest with all new jobs found

## 🛠️ Tools & Technologies
| Tool | Purpose |
|------|---------|
| n8n | Workflow automation |
| RapidAPI | Live jobs data source |
| Google Sheets | Job database storage |
| Gmail | Daily email notifications |
